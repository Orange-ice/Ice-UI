## 开始使用

1. 添加 CSS 样式

使用本框架前，请在 CSS 中开启 border-box

```
*,*::before,*::after{box-sizing: border-box;}
```

IE 8 及以上浏览器都支持此样式。

2. 安装 ice-vue-ui

```
npm i --save ice-vue-ui
```

3. 引入 ice-vue-ui

```
import {Button, ButtonGroup, Icon} from 'ice-vue-ui'
import 'ice-vue-ui/dist/index.css'

export default {
  name: 'app',
  components: {
    'g-button': Button,
    'g-icon': Icon
  }
}
```
